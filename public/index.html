vatcalc

Example:

  • http://mydomain.com/sc - yields the tax rate for South Carolina
  • http://mydomain.com/gb - yields the vat rate for Great Britain
  • http://mydomain.com/jp?subtotal=100.50 - you can optionally calculate the tax for the specified region

Responses are formatted as JSON, e.g.

{ rate: 0.08, location: "Florida", locationAbbreviated: "FL" }

Or, when calculating the tax

{ rate: 15.00, location: "Australia", locationAbbreviated: "AUS", subtotal: 100.00, tax: 15.00, }
